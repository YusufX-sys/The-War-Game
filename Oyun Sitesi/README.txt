Oyunun oynanması için Python indirilmeli ve komut isteminde (CMD) "pip install ursina" yazıp Ursina kütüphanesi indirilmelidir. Python indirme linki: https://www.python.org/downloads/
To play the game, you need to install Python and then open Command Prompt (CMD) and type "pip install ursina" to install the Ursina library.  
Python download link: https://www.python.org/downloads/
